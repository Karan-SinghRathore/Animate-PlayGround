Getting Started

What is CSS Animation?

CSS animations allow you to change CSS properties smoothly over time. They're powerful tools for creating engaging user experiences without JavaScript.

Step-by-Step: Your First Animation

1. Copy Animation Code

Go to our Gallery and click "Copy Code" on any animation you like.

2. Create HTML Structure

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Animation</title> <link rel="stylesheet" href="styles.css"> </head> <body> <div class="animated-element"> Hello, Animation! </div> </body> </html>

3. Add CSS File (styles.css)

/* Paste the copied animation here */ @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-30px); } } .animated-element { width: 100px; height: 100px; background: linear-gradient(45deg, #00e5ff, #00bcd4); border-radius: 10px; animation: bounce 2s infinite; display: flex; align-items: center; justify-content: center; color: white; font-weight: bold; }
Pro Tip: Always test your animations in different browsers and on mobile devices to ensure compatibility!

Basic Concepts

The @keyframes Rule

This defines what happens during the animation. You specify styles at different percentages of the animation timeline.

@keyframes animationName { 0% { /* Start of animation */ transform: scale(1); opacity: 1; } 50% { /* Middle of animation */ transform: scale(1.2); opacity: 0.7; } 100% { /* End of animation */ transform: scale(1); opacity: 1; } }

Animation Properties

  • animation-name: Links to your @keyframes (e.g., "bounce")
  • animation-duration: How long the animation takes (e.g., "2s")
  • animation-timing-function: Speed curve (ease, linear, ease-in-out)
  • animation-delay: Wait time before starting (e.g., "0.5s")
  • animation-iteration-count: How many times to repeat (1, 3, infinite)
  • animation-direction: Play direction (normal, reverse, alternate)

Shorthand Syntax

/* Long form */ animation-name: bounce; animation-duration: 2s; animation-timing-function: ease-in-out; animation-iteration-count: infinite; /* Shorthand (same thing) */ animation: bounce 2s ease-in-out infinite;

üîß Implementation Guide

Method 1: External CSS File (Recommended)

Create a separate CSS file for better organization:

/* Your copied animation */ @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-30px); } } /* Apply to any element */ .my-button { animation: bounce 1s infinite; /* Other styles */ }
<link rel="stylesheet" href="styles.css"> <button class="my-button">Bouncing Button</button>

Method 2: Internal CSS

<head> <style> @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-30px); } } .animated-element { animation: bounce 2s infinite; } </style> </head>

Method 3: Inline Styles (Not Recommended)

Warning: You cannot use @keyframes in inline styles. This method requires JavaScript.

Customization Tips

Timing Functions Explained

  • ease: Slow start, fast middle, slow end (default)
  • linear: Same speed throughout
  • ease-in: Slow start, then faster
  • ease-out: Fast start, then slower
  • ease-in-out: Slow start and end
/* Custom cubic-bezier curve */ animation-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55);

Animation Direction Options

animation-direction: normal; /* 0% ‚Üí 100% */ animation-direction: reverse; /* 100% ‚Üí 0% */ animation-direction: alternate; /* 0% ‚Üí 100% ‚Üí 0% ‚Üí 100% */ animation-direction: alternate-reverse; /* 100% ‚Üí 0% ‚Üí 100% ‚Üí 0% */

Multiple Animations

.element { animation: bounce 2s infinite, fadeIn 1s ease-out, colorChange 3s infinite alternate; }
Creative Tip: Combine multiple simple animations to create complex effects!

üêõ Troubleshooting

Common Issues & Solutions

Animation Not Working

Problem: The animation doesn't play at all.

Solutions:

  • Check that your @keyframes name matches the animation-name
  • Ensure you have both @keyframes and animation property
  • Verify CSS syntax (missing semicolons, brackets)
  • Check browser developer tools for errors

Animation Too Fast/Slow

/* Too fast? Increase duration */ animation: bounce 4s infinite; /* was 1s */ /* Too slow? Decrease duration */ animation: bounce 0.5s infinite; /* was 2s */

Animation Jumps at the End

This happens when your 0% and 100% keyframes don't match:

@keyframes smooth-bounce { 0%, 100% { transform: translateY(0); /* Same values */ } 50% { transform: translateY(-30px); } }

Animation Only Plays Once

/* Add 'infinite' to repeat forever */ animation: bounce 2s infinite; /* Or specify number of times */ animation: bounce 2s 3; /* plays 3 times */

Browser DevTools Debugging

  1. Right-click on animated element ‚Üí "Inspect Element"
  2. Go to "Computed" tab to see applied animations
  3. Use "Animations" panel to control playback
  4. Check "Console" for any CSS errors

Performance Tips

Use GPU-Accelerated Properties

These properties trigger hardware acceleration for smooth 60fps animations:

Fast Properties: transform, opacity, filter
Slow Properties: width, height, top, left, background-color
/* GOOD - Uses transform */ @keyframes slide-good { from { transform: translateX(-100px); } to { transform: translateX(0); } } /* BAD - Changes layout property */ @keyframes slide-bad { from { left: -100px; } to { left: 0; } }

Optimize for Mobile

  • Use shorter durations on mobile (1-2s max)
  • Reduce the number of simultaneous animations
  • Add will-change property for complex animations
  • Test on actual devices, not just desktop
.animated-element { will-change: transform; /* Hint to browser */ animation: bounce 1.5s ease-out; /* Shorter duration */ } /* Remove will-change when done */ .animated-element.finished { will-change: auto; }

üåê Browser Support

Modern Browser Support

CSS animations work in all modern browsers:

  • Chrome 43+ (2015)
  • Firefox 16+ (2012)
  • Safari 9+ (2015)
  • Edge 12+ (2015)

Legacy Browser Support

Internet Explorer: Partial support in IE10+, no support in IE9 and below.

Vendor Prefixes (Rarely Needed Now)

/* Usually not needed anymore, but for very old browsers: */ @-webkit-keyframes bounce { /* Old Chrome/Safari */ } @-moz-keyframes bounce { /* Old Firefox */ } @keyframes bounce { /* Modern browsers */ }

Feature Detection

// Check if CSS animations are supported if ('animation' in document.documentElement.style) { // Animations supported - add animated class element.classList.add('animated'); } else { // No animation support - show static version element.classList.add('static'); }

Example 1: Animated Loading Button

<button class="loading-btn" onclick="startLoading()"> <span class="btn-text">Click Me</span> <span class="spinner hidden"></span> </button>
.loading-btn { padding: 12px 24px; border: none; border-radius: 8px; background: #00bcd4; color: white; cursor: pointer; position: relative; transition: all 0.3s ease; } .spinner { width: 20px; height: 20px; border: 2px solid transparent; border-top: 2px solid white; border-radius: 50%; animation: spin 1s linear infinite; } .hidden { display: none; } @keyframes spin { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } }
function startLoading() { const btn = document.querySelector('.loading-btn'); const text = btn.querySelector('.btn-text'); const spinner = btn.querySelector('.spinner'); text.classList.add('hidden'); spinner.classList.remove('hidden'); btn.disabled = true; // Simulate loading setTimeout(() => { text.classList.remove('hidden'); spinner.classList.add('hidden'); btn.disabled = false; text.textContent = 'Success!'; }, 3000); }

Example 2: Card Hover Effects

<div class="card"> <img src="image.jpg" alt="Card image"> <div class="card-content"> <h3>Card Title</h3> <p>Card description text</p> </div> </div> <style> .card { width: 300px; border-radius: 12px; overflow: hidden; box-shadow: 0 4px 8px rgba(0,0,0,0.1); transition: all 0.3s ease; cursor: pointer; } .card:hover { transform: translateY(-8px) scale(1.02); box-shadow: 0 12px 24px rgba(0,0,0,0.15); } .card img { width: 100%; height: 200px; object-fit: cover; transition: transform 0.3s ease; } .card:hover img { transform: scale(1.1); } .card-content { padding: 1.5rem; background: white; } </style>
